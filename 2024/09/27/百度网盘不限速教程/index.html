——————————————————— 百度网盘不限速 ———————————————————————

这应该算是好物分享了吧。毕竟我也没学过易语言，不过他全是中文有点其他编程基础的都很好理解。这个软件还是最近才发布的，我看了一下用的是确实是易语言。真是造福国人。c 的话有时间我用 c 也写一个加速脚本，源码放在 github 上，今天就看看别人的易语言吧。

首先关注他的微信公众号 it 技术世界，发 20412 获取旧版网盘安装包，上面说的很清楚可能到时候你看到的下载速度只有几 kb 但是相信下载文件的时候你会发现那速度很离谱。具体易语言可能以后会接触吧，有兴趣可以自己去了解。要是过一段时间那公众号被举报不在了的话，加我 qq3059668258。这边存了一份，一般来说新版修复的都差不多了，也不会管旧版，白嫖就完事了。
