c++/java学习总结

Created2024-05-30|Updated2024-05-30
|Post Views:

学习方向的选择

自此学习完了 c 之后，我感觉自己无所不能。想着创建一个自己完全写的外挂。但首先第一步我想把这个东西给放在安卓平台上来用，我真的是脑袋都大了，除了在 VS 上敲代码后什么都不会。网上找了大量的相关资料，发现 java 在图形界面的创建还是幺幺领先的。但我并不喜欢 java，想着用 c# 去代替它，等我用来 inj 和 andriod studio 才发现还是 java 好用点，各种语言直接的逻辑大体都差不多，只是文件格式不一样。想起前段时间的前端，vm 虚拟机的学习是痛苦的，一会相当渗透工程师一会想当 ue5 游戏地编，更有所想想去当码农。。。可恶，我也明白东西学的太杂，什么都干不好。除非脑袋大，什么都会。。。犹豫了片刻，还是打算在游戏方面侧重点发展，精通一门 c 就够了，其他语言能看懂就行。

安卓 app 的制作

这种很简单，用 andriod studio 新建项目 -> 选择空项目 -> 新建安卓项目，然后真机调试，手机打开开发者模式，用数据线链接电脑。因为 java 是在虚拟机里面运行的，也可以在你本机上运行，所以你可以这样然后那样，就可以点击运行，他会在你手机里安装包，里面就是你 java 编写的内容，总体只要会应用，毫无操作可言。

c++ 代码打包成 exe

如果你要打包 app，那当我无话可说，你要使用第三方软件是必然的。如果你要打包 exe 那么可以用它自带的打包软件，对了，这是 visualstudio 的打包方法，安卓小插件，搜 instuall 就都出来了，点第一个就好，重启点击这个项目的资源管理器，新建 set… 然后资源文件，左边 app… 那一行右键 add 你的项目，然后在右键选择主输出确定，重新生成代码安装，就好了。如果你不想你的源码公布那就把源码删了再分享。
