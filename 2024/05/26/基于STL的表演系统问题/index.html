第一个：1 > 菜单.obj : error LNK2019: 无法解析的外部符号 “public: void __cdecl preson::caidan (void)” (?caidan@preson@@QEAAXXZ)，函数 main 中引用了该符号。

像这种问题造成的原因是因为，你在你头文件中类的创建里面定义了一个函数声明，而你在对接 cpp 文件中没有加上函数作用域，还有一种问题是因为你文件中有两个 cppc 文件重复包含 main 函数，删除一个重新生成就好。
